## 🚀 Featured work

### [MindMate](https://github.com/MindMate-Project) — Alzheimer's care & patient monitoring platform
A five-component system (backend, web, mobile, AI microservice, IoT). I architected the **Node.js/TypeScript backend** that connects to a Dockerized **Python/FastAPI face-recognition service**, with real-time patient tracking over Socket.IO, ESP32 hardware integration, CRON-based medication scheduling, and Firebase push notifications.

### [Online Shop](https://github.com/mohammedhossam86/Online-Shop) — E-commerce backend
RESTful backend with product catalogs, sessions, and cart operations. MongoDB schema design, JWT auth, and role-based access control.

### 🦊 GitLab open source contributions
Core contributor to `gitlab-org/gitlab` — merged MRs improving platform stability, API documentation, CI/CD pipelines, and Vue components. Awarded multiple Contributor Rewards by GitLab's Developer Relations team.
